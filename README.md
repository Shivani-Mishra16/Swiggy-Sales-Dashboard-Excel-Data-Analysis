# Swiggy-Sales-Dashboard-Excel-Data-Analysis

**Overview**
This project focuses on analyzing Swiggy food delivery data using Microsoft Excel to build an interactive dashboard and extract meaningful business insights.
**Objectives**
Analyze sales performance and customer trends.
Track key performance indicators (KPIs).
Build an interactive dashboard for business insights.
**Dataset**
Food delivery dataset containing order details, ratings, categories, and locations.
**Total records analyzed: 197K+**
**Tools & Technologies**
Microsoft Excel
Pivot Tables & Pivot Charts
Slicers & Dashboard Design
Data Cleaning & Transformation
**Key Features**
**KPI Metrics:** Total Orders, Average Rating, Average Order Value
**Sales Trends:** Daily, Weekly, Monthly analysis
**Category Analysis:** Veg vs Non-Veg sales distribution
**Geographic Insights:** State-wise performance
Top cities analysis
Interactive filters (Slicers)
**Key Insights**
Identified peak sales periods and high-demand days.
Highlighted top-performing cities contributing to revenue.
Observed customer preferences across food categories.
Discovered regional variations in sales performance.
**Project Outcome**
This project demonstrates strong skills in:
Data Analysis
Data Visualization
Dashboard Development
Business Insight Generation
